Condicionais

Sentenças lógicas e expressões condicionais :

As expressões condicionais em Javascript podem ser escritas com as seguinte expressão geral (EXP OP EXPP) link (EXP OP EXPP) ou por exemplo (score >= minimo), (pontuação == rifa) && (dia <= target).

Os operadores lógicos são && : e, || : ou, ! : not. Também temos os operadores tradicionais == : igual, != : diferente, === : estritamente igual, !== : estritamente diferente, > : maior que, < : menor que, <= : menor ou igual, >= : maior ou igual.

Sentenças do tipo else if:

Sentenças desse tipo são usadas para filtar oque acontece a depender do desenrolar do código. No javascript o if pode ser usado indenpendente do else, com sendo falsa a proposição dentro do if, levando ao código apenas continuar sendo lido pelo interpretador (ignorando oque está dentro do IF). A sintaxe do if em javascript é a seguinte :

if ( a op b) {
conteudo ;
}

exmeplo prático:

function parabens(){
console.log('parabéns');
}

if (score >= min) {
parabens();
}

Sentenças com else

Quando você coloca o else você faz com que ele execute algo quando o valor da expressão lógica no if é de falso. A sintaxe do else/if e um exemplo prático seguem abaixo, junto com a saida de um programa executado no interpretador:

if ( a op b) {
conteudo ;
} else {
conteudo2 ;
}

exemplo prático

function parabens(){
console.log('parabéns');
}

function fail(){
console.log('fail');
}
if (score >= min) {
parabens();
} else {
fail();
}

Sentenças com switch

As sentenças com switch são análogas ao elif do python. Elas são praticas de serem usadas quando apenas um conjunto discreto e finito de valores para uma dada variável importam para decidir que código vai rodar. Para algo fora desse conjunto usa-se a opção default. A sintaxe do switch é da segunte forma, logo abaixo vem a saida de um script com ele.

var id = valor ;
switch(id){
case 1 :
document.write('1');
break ;

case 2 :
document.write('2');
break ;

case 3 :
document.write('3');
break ;

default :
document.write('Nenhum desses');
break;

}

Loops

Loop for

O loop for é usado quando é conhecido o número de iterações que o loop deseja dar. Para usar esse loop parte de oque escrevemos é for (exp) essa ecpressão exp pode ser sempre divida como : (inicializador da variável, condição para que o loop repita, argumento de incrementação da variável). Sintaxe com exemplo :

for (var i = 0; i<=10 ; i++){
document.write(i);
}
